### 2.2 AI生成内容特别规定（已核实）

**2024.04.02 公告核心内容**：
- 禁止利用AI生成虚拟人物进行虚假营销
- 禁止AI生成内容冒充真实人物
- 需要明确标识AI生成内容

**2023.05.09 AI内容标识规范**：
- AI生成的内容需要添加水印标识
- 需要在元数据中标注AI生成
- 不得利用AI生成内容从事违法违规行为

### 2.3 禁止内容（已核实）

| 类别 | 禁止表述 | 违规等级 |
|------|----------|----------|
| 荐股 | 「买XX股」「建仓XX」「这股必涨」 | 🔴 严重 |
| 预测 | 「明天大盘会上XX点」 | 🔴 严重 |
| 承诺收益 | 「跟我做稳赚」「保证收益」 | 🔴 严重 |

| AI虚拟人冒充真人 | 利用AI虚拟人设骗取信任 | 🔴 严重 |

### 2.4 安全内容类型（已核实）

| ✅ 可以做 | 说明 |
|----------|------|
| 概念科普 | MACD、K线、财报怎么看 |
| 原理讲解 | 为什么追涨容易亏钱 |

### 2.5 风险提示（必须）

每条视频结尾需要：
> 「以上内容仅为个人学习分享，不构成任何投资建议。投资有风险，决策需谨慎。」

### 2.6 违规处罚（已核实案例）

| 违规次数 | 处罚 |
|----------|------|
| 第1次 | 警告+限流 |
| 第2次 | 限流+降低权重（1-4周） |
| 第3次 | 封禁7天 |
| 严重违规 | 直接封禁 |

**已公布的处置类型**：
- 无限期封禁（针对恶劣违规）
- 封禁主播/账号公告（定期发布）

## 五、视频号（微信生态，已核实）

### 5.1 重要条款（4.6.2）- 禁止财经内容引流

**4.6.2** 是视频号最关键的财经相关条款：

> "未经相关部门许可，进行**荐股**、**推荐金融产品**等相关投资咨询活动（包括但不限于证券、期货、基金等）。或者以分享财经信息（如**股市分析/行业分析**、**投资技巧**等）为由将用户**诱导或引流**进行推广营销。"

这意味着：
- ❌ 荐股（无论是否收费）→ 禁止
- ❌ 推荐金融产品 → 禁止
- ❌ 股市分析/行业分析内容引流推广 → 禁止
- ❌ 投资技巧内容引流 → 禁止

**注意**：与抖音快手不同，视频号连"股市分析"分享本身都可能被认为有问题（如果导致引流推广）。

### 5.2 视频号禁止内容（与抖音快手一致）

| 类别 | 说明 |
|------|------|
| 荐股 | 禁止 |
| 虚假信息 | 谣言、内幕 |
| 诱导交易 | 私信/加群/转账 |
| 仿冒他人 | 冒充专业人士 |
| 虚假宣传 | 夸大收益 |

### 5.3 视频号安全内容

| ✅ 可以做 | 说明 |
|----------|------|
| 概念科普 | 投资理论、金融知识 |
| 原理讲解 | 为什么某策略有效 |
| 自我经历 | 我踩过什么坑（不提具体标的） |
| 风险提示 | 每条内容结尾必须有 |
| 纯知识分享 | 不带任何引流意图 |

### 5.4 特殊注意事项

视频号在微信生态下，有额外风险：
- **熟人传播**：内容容易被朋友圈/群转发，违规传播更快
- **举报机制**：容易被熟人举报
- **关联影响**：违规可能影响微信支付等功能

### 5.5 违规处罚

微信有权：
- 警告
- 删除内容
- 限制功能
- 封禁账号
- 禁止重新注册

## 八、发布前检查清单

> 适用于所有平台

### 8.1 文案检查

```
□ 没有具体股票名称/代码
□ 没有预测具体点位/收益率
□ 没有「稳赚」「必涨」「保证」等绝对词
□ 没有诱导行为（私信/加群/转账）
□ 没有虚假信息（内幕/庄家）
□ 有风险提示语句
□ 定位是「知识分享」而非「投资建议」
□ 没有利用AI虚拟人冒充真人
```

### 8.2 画面检查

```
□ 没有股票代码
□ 没有真实K线截图
□ 没有真实收益率数据（除非是教育用途的模拟数据）
□ 图表使用模拟数据
□ 有AI生成标识（如使用了AI画面）
□ 没有其他违规元素
```

### 8.3 标题+简介检查

```
□ 标题没有承诺收益
□ 简介没有荐股暗示
□ 话题标签没有敏感词
□ @提及没有诱导内容
□ 有AI生成内容标识
```
