# hermes-backup

Hermes Agent 创作流水线备份仓库

## 内容

- `/skills/` - 所有技能定义（SKILL.md）
- `/workflow/` - 工作流状态和配置
- `/logs/` - AI 行为日志

## 结构

```
hermes-backup/
├── README.md
├── skills/
│   ├── creative/           # 创作相关技能
│   ├── social-media/       # 社交媒体合规
│   ├── browser-automation/  # 浏览器自动化
│   └── ...
├── workflow/
│   ├── workflow_state.json
│   └── config.json
└── logs/
    └── ai_actions.log
```

## 更新说明

此仓库由 Hermes Agent 自动维护，保存创作流程的关键状态。
